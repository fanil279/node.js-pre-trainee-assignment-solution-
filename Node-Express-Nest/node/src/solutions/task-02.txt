Task 02 – Stream Transformer

1 - Description

I implemented a CSV stream transformation pipeline using Node.js streams.

The implementation includes:

* A custom `CSVParser` stream that reads CSV data and converts each row into a JavaScript object.
* A `DataTransformer` stream that normalizes and transforms each record:
  * Capitalizes names and cities.
  * Normalizes email addresses.
  * Formats phone numbers.
  * Standardizes dates to the `YYYY-MM-DD` format.
* A `CSVWriter` stream that converts transformed objects back into CSV format.
* A `processCSVFile()` function that connects all streams using `pipeline()` to process CSV files efficiently.
* A `createSampleData()` function that creates the sample `data/users.csv` file for testing.

The implementation processes the file using streams, allowing records to be transformed without loading the entire file into memory.


2 - How to Run

1. Make sure Node.js is installed
2. Clone the repository
3. Install dependencies with npm install
4. Open a terminal in the project directory at: Node-Express-Nest\node\src\solutions
5. Run the implementation with: node task-02.js

This will process data/users.csv and generate the transformed output in data/users_transformed.csv.
