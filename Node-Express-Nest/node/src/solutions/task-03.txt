Task 03 – Event Loop Analysis & Async Debugging

1 - Description

I implemented the required Event Loop analysis and asynchronous debugging tasks.

The implementation includes:

* Analysis of all six Node.js event loop phases:

  * Timers
  * Pending Callbacks
  * Idle/Prepare
  * Poll
  * Check
  * Close Callbacks
* Explanation of the execution order of synchronous code, microtasks (`process.nextTick()` and `Promise.then()`), and macrotasks.
* Prediction of the execution order for the provided asynchronous code snippets with explanations.
* Fix for the race condition in file processing using `async/await` and `Promise.all()`.
* Conversion of callback hell into readable `async/await` code with centralized error handling.
* Replacement of mixed Promise and callback code with a consistent `async/await` implementation.
* Demonstration of the Node.js event loop using `setTimeout()`, `setInterval()`, `setImmediate()`, `fs.readFile()`, `process.nextTick()`, Promise microtasks, and stream close callbacks.
* Utility functions for creating test files and logging execution by event loop phase.


2 - How to Run

1. Make sure Node.js is installed
2. Clone the repository
3. Install dependencies with npm install
4. Open a terminal in the project directory at: Node-Express-Nest\node\src\solutions
5. Run the implementation with: node task-03.js
